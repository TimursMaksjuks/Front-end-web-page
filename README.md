# Front-end-web-page
This project is a fan-made web page dedicated to Kanye West, created as a front-end practice project. 
The page includes custom styling, images, and a mini quiz for users. JavaScript is used to handle user interaction, collect quiz answers, and send the submitted responses through a link. 
The quiz form submits data to a demo external endpoint as part of the educational project requirements.
The project was built with HTML, CSS, and JavaScript to practice webpage structure, styling, and basic form interaction.
